# Agility CMS Plans

Agility CMS offers fixed-price plans that scale with team size and number of sites, not API call volume. All plans include unlimited API requests, content models, and locales.

Source: https://agilitycms.com/product/pricing

## Starter — $1,425/month (annual billing)

For a single website and small teams.

- Users: 10
- Content Entries: 50,000
- Page Sitemaps: 1
- Asset Storage: 1 TB
- Bandwidth: 1 TB/month
- SLA: None
- Features: Unlimited API requests, content models, locales, page management, rich text, versioning, workflows, DAM, CDN

## Pro — $2,850/month (annual billing)

Most popular. Ideal for larger teams, multiple websites, and content hubs.

- Users: 25
- Content Entries: 250,000
- Page Sitemaps: 5
- Asset Storage: 5 TB
- Bandwidth: 2 TB/month
- SLA: Available
- Features: Everything in Starter, plus custom roles, teams management, MFA, SSO

## Enterprise — $5,000/month (annual billing)

At scale, with multiple properties and granular control over every detail.

- Users: 50+
- Content Entries: 500,000+
- Page Sitemaps: 10+
- Asset Storage: 20 TB
- Bandwidth: 15 TB+/month
- SLA: 99.9%+
- Features: Everything in Pro, plus dedicated success manager, private Slack channel, custom security reporting, advanced compliance controls

## Free Tier

Agility CMS also offers a free tier that includes API access and can power a production solution, suitable for small projects or evaluation.

- Pricing: $0
- API Access: Yes
- Production Use: Allowed
