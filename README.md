# Agility CMS

Enterprise headless CMS for marketing and developer teams. Agility CMS delivers content via REST and GraphQL APIs with CDN-backed global delivery, page management, multi-site support, and SDKs for Next.js, Gatsby, Nuxt, Astro, and .NET.

- **Website**: https://agilitycms.com/
- **Documentation**: https://agilitycms.com/docs
- **Pricing**: https://agilitycms.com/product/pricing
- **GitHub**: https://github.com/agility
- **LinkedIn**: https://www.linkedin.com/company/agilitycms

## APIs

- **Content Fetch API (REST)**: `https://api.aglty.io/{guid}/{apitype}` — fetch content items, lists, pages, and sitemaps over CDN
- **GraphQL API**: Instance-specific endpoint accessible via Settings > API Keys in the Agility dashboard
- **Content Sync API**: Bulk content sync for build-time (SSG) workflows
- **Content Management API**: Programmatic content and asset creation/updates
